Project Overview

WebLang is a custom web language allowing developers to write .wlf files for HTML content and .swlf files for styling, which are compiled automatically to standard HTML and CSS. The project aims to simplify website creation while retaining full HTML/CSS capabilities.

The compiler supports multiple commands and attributes in English, all written in a single line without mandatory indentation, making it highly readable and fast to write.

Main Features

  • Simple Syntax

    One command per line. Everything in English. No indentation needed.

  • Full HTML Support

    All standard HTML tags and attributes can be used via commands.

  • Style Files

    SWLF files compile to CSS and are automatically linked to the HTML.

  • Live Compilation

    The compiler can run automatically, generating output instantly.

  • Extensible

    Add new commands and attributes easily, everything is modular.

  • Export Ready

    Compile to standard HTML/CSS and deploy on any server.

How To Use

Example WLF

page title="My WebLang Site" lang="en" charset="UTF-8"
section id="main" text="Welcome to my WebLang site!" class="main"
button text="Click me" link="about.wlf" class="btn"
footer text="© 2025 WebLang Project" class="footer"

Example SWLF

body background="#0b1220" color="#e6eef6" font-family="Arial, sans-serif" margin="0" padding="20px"
section.main padding="60px" text-align="center" background="#071022" border-radius="12px"
a.btn background="#66d9ef" color="#071022" padding="12px 20px" border-radius="8px" text-decoration="none" font-weight="700"
a.btn:hover background="#8a6cff"

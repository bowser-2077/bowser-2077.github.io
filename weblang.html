Key Features

Simple Syntax

Write your site in one line per command, all in English. No indentation required.

Full Web Support

Supports all standard HTML elements and CSS properties, fully compiled to compatible web output.

Automatic Styling

Create SWLF style files that compile into CSS and link automatically to your pages.

Live Compilation

Compile your project instantly and preview changes live with minimal setup.

Extensible

Add custom commands or modules easily. WebLang adapts to your project needs.

Deploy Ready

Output clean HTML and CSS files ready to host on any server or platform.

Getting Started

1. Download the WebLang compiler from the official repository.

2. Create your WLF pages and optional SWLF style files.

3. Run the compiler to generate standard HTML and CSS.

4. Open your site in any browser and see your changes instantly.
